# Bank-Customer-Churn-Intelligence-Dashboard
Designed and developed a multi-page Power BI and Python analytics dashboard tracking 10,000 customers to evaluate churn behavior and identify retention opportunities.

## 🚨 The Problem
A retail bank is facing unmitigated, silent customer attrition. Lacking automated risk alerts and data-driven indicators, relationship teams remain reactive, leaving the bank blind to structural drivers and losing revenue.

## 📉 The Effect
This accelerating churn heavily compromises the bank's financial stability, draining high-value lifetime value (LTV) customers and driving up customer acquisition costs (CAC). Beyond direct revenue loss from shrinking deposit balances, unmanaged attrition erodes market share to digital competitors and damages institutional brand health. Operationally, relationship management teams waste critical resources executing blanket, generalized retention campaigns that yield poor conversion rates because they lack precise target profiles.

## 🛠️ The Solution
This multi-page Power BI and Python analytics dashboard tracks 10,000 customers to transform raw operational data into actionable retention strategies. Integrating exploratory data analysis (EDA) with predictive risk segmentation, the platform classifies customers by vulnerability tier, credit scores, and engagement velocity. The system introduces dynamic risk matrix tracking, giving account teams an automated early-warning framework to deploy targeted, high-conversion retention offers and structural fixes before high-value segments churn.

## ⚙️ Process
Leveraged Python to import, clean, and filter raw data, executing outlier detection and removing irrelevant columns. The optimized dataset was then imported into Power BI for comprehensive data modeling, DAX engineering, and multi-page visual dashboard development.

## 📁 Data Source & Project Architecture

### 📊 Dataset Source
The analysis is built on a dataset tracking 10,000 retail banking customers. 
* **Source:** [Kaggle Bank Customer Churn Dataset](https://www.kaggle.com/datasets/gauravtopre/bank-customer-churn-dataset)

---

### 🗺️ File Navigation Guide

* **`data/`**
  * `Customer-Churn-Records.csv`: The original, unaltered source dataset containing 10,000 customer records.
* **`notebooks/`**
  * `BankCustomerChurnDataSetQualityCheck.ipynb`: Jupyter notebook containing the Python script for data sanity checks, missing value handling, outlier detection, and column filtering.
* **`dashboard/`**
  * `BankCustomerChurnDataDashboard.pbix`: The complete multi-page Power BI dashboard file containing the data model, DAX measures, and visual layouts.
